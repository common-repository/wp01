== Installation ==
For automatic installation:
go to the WordPress admin panel under "Plugins" - > "Add New", then type " WP01 "and click"Install".

Manual installation is also very easy:
1. Download the zip archive from the site and unzip it
2. Upload the "wp01" plugin folder to the "/wp-content/plugins" directory/"
3. Activate the plugin via the Plugins menu in WordPress
4. Select "Improvements from WP01" in the panel and make the necessary improvements to your site!
